# Lab 6 — Local API server (optional)

В задании Lab 6 используется `fetch()` и нужен сервер (API), который отдаёт:

- `GET /images` → JSON массив изображений
- `POST /temperature` → принимает JSON и возвращает JSON с сообщением

Если у вас уже есть сервер от преподавателя ("сервер с документацией"), этот раздел не нужен.
Если сервера нет — используйте этот простой mock-сервер локально.

## Запуск

Перейдите в папку `server` и выполните:

```bash
npm i
npm start
```

Сервер поднимется на `http://localhost:3000`.

## Подключение фронтенда

В `lab6.html` укажите:

```html
<body data-api-base="http://localhost:3000">
```

После этого галерея и отправка температуры начнут работать.

## Замечание про Render

Если вы хотите, чтобы всё работало и на хостинге, можно деплоить **не Static Site**, а **Web Service (Node.js)**,
чтобы этот сервер обслуживал и статические файлы, и API.
